## What I Build

I engineer **end-to-end AI media production systems** — fully automated pipelines that go from a single topic prompt to a published YouTube video with zero manual intervention. My work sits at the intersection of large language models, generative audio/image/video synthesis, and production-grade cloud infrastructure.

Current output target: **300 minutes of video per day**, fully automated.
