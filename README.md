## What I Build

I run **[InFrontWebs](https://infrontwebs.com)** — a UK-registered AI automation and cloud infrastructure firm. My focus is on building end-to-end media production systems that replace manual workflows with fully automated pipelines.

- **Video automation at scale** — LLM-scripted, TTS-narrated, AI-illustrated documentary content produced and uploaded to YouTube automatically. Targeting 300 min/day output.
- **AI voice & image pipelines** — Chatterbox TTS with voice cloning, Flux AI image generation on RunPod GPU, MuseTalk talking-head synthesis.
- **Cloud & DevOps** — GitHub Actions CI/CD, RunPod GPU orchestration, serverless infra.
